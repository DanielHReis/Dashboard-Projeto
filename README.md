# Dashboard de Salários em Data Science

## Conjunto de Dados
Fonte: [Data Science Salaries Dataset](https://www.kaggle.com/datasets/ruchi798/data-science-job-salaries) (exemplo) no Kaggle.

## Justificativa da Escolha
Este conjunto de dados foi selecionado porque aborda um tema de alta relevância no mercado de tecnologia: a remuneração na área de dados. Ele permite responder perguntas como:
- Quais cargos em Data Science são mais bem remunerados?
- Como a experiência (Júnior, Pleno, Sênior) impacta o salário?
- O trabalho remoto influencia os salários?
- Como os salários evoluíram de 2020 a 2022?

Essas questões são de grande interesse para profissionais da área, estudantes e empresas, tornando o dashboard uma ferramenta prática e informativa.
